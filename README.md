# South-African-Unemployment-Analysis

## 📊 Overview
This project analyzes unemployment trends in South Africa using **Stats SA Quarterly Labour Force Survey (QLFS)** data.  
Latest release (Q1 2026): National unemployment rate **32.7%**, youth unemployment **60.2%**.

## 📂 Contents
- `data/unemployment.csv` → dataset
- `notebooks/unemployment_analysis.ipynb` → analysis notebook
- `images/` → charts & dashboards

## 🔑 Insights
- Unemployment rose from **31.4% (Q4 2025)** to **32.7% (Q1 2026)**.
- Youth unemployment remains critically high at **60.2%**.
- Labour force participation dropped to **59%**, lowest since 2022.

## 🛠️ Tools
- Python (pandas, matplotlib, seaborn)
- Power BI (optional dashboard)
- Excel (optional pivot tables)
